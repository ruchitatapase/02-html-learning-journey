
# Headings and Paragraphs

## Project

Spotify Landing Page (Basic Version)

## Topics Covered

- `<title>` – Displays the webpage title in the browser tab.
- `<h1>` – Defines the main heading of the webpage.
- `<h2>` – Defines a secondary heading.
- `<p>` – Displays paragraph text.

## What I Learned

Today I learned how to use headings and paragraphs to organize webpage content. I recreated a simple Spotify-inspired landing page using only the HTML tags I have learned so far.
