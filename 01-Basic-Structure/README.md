# Basic HTML Structure

## Topics Covered

- `<!DOCTYPE html>` – Declares that the document is an HTML5 document.
- `<html>` – The root element that contains all the content of the webpage.
- `<head>` – Contains information about the webpage, such as the title and metadata. This information is not displayed on the webpage.
- `<title>` – Specifies the title of the webpage that appears on the browser tab.
- `<body>` – Contains all the visible content of the webpage, such as text, images, links, and other elements.
