
# HTML Text Formatting

## 📚 Topics Covered

- `<b>` – Displays text in bold.
- `<strong>` – Highlights important text.
- `<i>` – Displays italic text.
- `<em>` – Emphasizes text (usually italic).
- `<u>` – Underlines text.
- `<mark>` – Highlights text.
- `<small>` – Displays smaller text.
- `<del>` – Shows deleted text.
- `<ins>` – Shows inserted text.
- `<sub>` – Displays subscript text.
- `<sup>` – Displays superscript text.
- `<br>` – Inserts a line break.
- `<hr>` – Inserts a horizontal line.
- `<a>` – Creates a hyperlink.

## 💻 Mini Project

### Spotify Premium Landing Page

A Spotify-inspired webpage created using HTML text formatting tags to organize and highlight important content.

## 🎯 What I Learned

- Format text using different HTML tags.
- Differentiate between `<b>` and `<strong>`.
- Differentiate between `<i>` and `<em>`.
- Use line breaks and horizontal rules to improve layout.
- Create hyperlinks using the `<a>` tag.
- Build a real-world webpage using only basic HTML elements.

## 🚀 Project Highlights

- Professional page title

- Feature list

---

## 📌 Key Takeaways

- Use formatting tags only when they add meaning or improve readability.
- Headings should organize sections, not simply enlarge text.
- Keep HTML code properly indented for better readability.
